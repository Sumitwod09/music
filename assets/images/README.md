# Image Assets

This directory contains image assets for the MusicFlow app.

## Required Images

- **app_icon.png** - Main app icon (512x512px)
- **app_icon_foreground.png** - Adaptive icon foreground (512x512px)
- **app_icon_background.png** - Adaptive icon background (512x512px)
- **placeholder_album.png** - Default album art placeholder

## Placeholder Images

You can use placeholder images from services like:
- https://picsum.photos/seed/musicflow/512/512
- https://via.placeholder.com/512x512/037171/FFFFFF?text=MF

## Instructions

1. Download or create the required images
2. Place them in this directory with the exact names specified above
3. Ensure images are optimized for mobile use (PNG format recommended)
