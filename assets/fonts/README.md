# Font Assets

This directory contains font files for the MusicFlow app.

## Required Fonts

- **Roboto-Regular.ttf** - Regular weight font for body text
- **Roboto-Bold.ttf** - Bold weight font for headings
- **Roboto-Medium.ttf** - Medium weight font for buttons

## Download Instructions

Download the Roboto font family from Google Fonts:
https://fonts.google.com/specimen/Roboto

Place the font files in this directory with the exact names specified above.
